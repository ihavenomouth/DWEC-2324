Ejercicios de interacción con el CSS

Ejercicio 3

Crea una web que contenga un cuadro de texto y un botón con la etiqueta Añadir que añada el contenido del cuadro de texto a una lista.

También habrá un botón con la etiqueta Colorear. Al pulsar este botón se colorearán los ítems de la lista con estilos en línea (con la etiqueta style) de la siguiente manera:

  • Los que tengan un número de caracteres impar tendrán un color de fondo rojo y letras en blanco.
  • Los que tengan un número de caracteres par tendrán un color de fondo golden-rod.
